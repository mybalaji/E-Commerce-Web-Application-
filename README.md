# E-Commerce-Web-Application-

## Overview
This is a responsive, multi-page E-commerce web application template designed using HTML, CSS, and Bootstrap. It features a simple navigation bar, product showcase with a carousel, registration, login, and orders page. The template is suitable for beginners looking to develop an E-commerce site or anyone who needs a quick start for a web project.

## Features
- **Home Page**: Includes a carousel slider showcasing products and featured product sections.
- **Registration Page**: Allows new users to register with a form.
- **Login Page**: Simple login form with validation.
- **Orders Page**: Displays past orders with item details.
- **Responsive Design**: Uses Bootstrap to ensure compatibility across all devices.
- **Interactive Elements**: Includes a simple alert on "Buy Now" buttons and form validation for login and registration.

## Pages
1. **index.html**: The main landing page with product categories and a featured product section.
2. **login.html**: User login page with validation.
3. **registration.html**: User registration page with form validation for password matching.
4. **orders.html**: Displays a list of previous orders made by the user.
5. **CSS Files**: External stylesheets (index.css, styles.css, orders.css) for custom styling.
